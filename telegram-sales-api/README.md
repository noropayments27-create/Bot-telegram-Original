# telegram-sales-api

API base para el bot de ventas en Telegram.

## Requisitos
- Node.js 18+
- npm

## Instalacion
```bash
npm install
```

## Variables de entorno
Crea un archivo `.env` basado en `.env.example`.

## Desarrollo
```bash
npm run dev
```

## Produccion
```bash
npm start
```

## Endpoint de salud
- GET `http://localhost:3001/health`
